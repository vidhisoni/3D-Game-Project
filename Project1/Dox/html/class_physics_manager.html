Detailed Description

Manages all contacts, generate list of Collisions/Triggers enter/exit/stay, gneerates Forces, updates positions, corrects Positions On Update constangly ask which objects are colliding, generates a list , and then. acts accordingly, helped by CollisionManager, to solve if some objects collide and what forces will be related, also how to react to specific set of collition Elements If any collider is trigger, there is not anyImpulse,.
