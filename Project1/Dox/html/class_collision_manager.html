Detailed Description

Manages all collisions2D, generates Forces, corrects Positions.

Helps PhysicsManager, calculating if objects collide, how far are they, how to react, this generates a list of Collisions, described on ContactPoints, and information related with the collision helped by the class Contact after physical reactions, it defines specifically reactions to specific Tupla of collision Elements
